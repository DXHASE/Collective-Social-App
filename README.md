  ## Description
  - A social media application that allows the user to create a profile, log in to that profile, and be able to create, update, and delete posts as well as comment on existing posts.

  ## Installation
  - The steps include: 

  npm init
  npm i
  express
  express-session
  express-handlebars
  dotenv
  mysql2
  sequelize
  bcrypt
  connect-session-sequelize
  eslint --save-dev
  npx eslint --init 
  .prettierignore
  .prettierrc.json
